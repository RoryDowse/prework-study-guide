HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The head element goes before the body element and is the container for metadata.
  • The header element is to be distinguised from the head element.
  • The header element goes in the body element and typically contains a heading element, logo, and/or authorship information.
  • The types of meta elements include character set, description, keywords, author, and viewport (amongst others).
  • The title element is also a form of metadata that is contained in the head element (though it does not have a meta tag).
  • Examples of semantic HTML include head, body, section, and footer (amongst others).

CSS

  • CSS rules are comprised of selectors which may hold more than one declaration.
  • Declarations are held in a "Declaration Block"
  • Each declaration contains a property and a value
  • The margin property indicates space around the outside of an element
  • The padding property adds space around content inside an element
